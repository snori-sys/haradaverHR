# VercelダッシュボードURL

## 🔗 Vercelダッシュボード

### メインダッシュボード
**URL**: https://vercel.com/dashboard

### プロジェクト作成ページ
**URL**: https://vercel.com/new

または、ダッシュボードから「Add New...」→「Project」をクリック

## 📋 アクセス方法

### ステップ1: ダッシュボードにアクセス

1. **ブラウザで以下にアクセス**:
   - https://vercel.com/dashboard

2. **ログイン**（まだログインしていない場合）
   - GitHub/GitLab/Bitbucketアカウントでログイン
   - または、Emailでサインアップ/ログイン

### ステップ2: プロジェクトを作成

1. **ダッシュボードの右上にある「Add New...」ボタンをクリック**
2. **「Project」を選択**

または、直接以下にアクセス：
- https://vercel.com/new

### ステップ3: プロジェクトをアップロード

1. **「Upload」タブをクリック**
2. **ZIPファイルを選択**
   - `point-card-pwa.zip` を選択
   - 場所: `/Users/sawashimanoriyuki/point-card-pwa/point-card-pwa.zip`

## ✅ 次のステップ

ダッシュボードにアクセスしたら：

1. ✅ **ログイン**
2. ✅ **「Add New...」→「Project」をクリック**
3. ✅ **「Upload」タブをクリック**
4. ✅ **ZIPファイルを選択**
5. ✅ **環境変数を設定**
6. ✅ **「Deploy」ボタンをクリック**

詳細は `Vercel手動アップロード手順.md` を参照してください。

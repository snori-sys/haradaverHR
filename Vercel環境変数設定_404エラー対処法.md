# Vercel環境変数設定 - 404エラー対処法

## ⚠️ 404エラーが発生した場合

直接URLでアクセスした際に404エラーが表示される場合、以下の原因が考えられます：

1. **プロジェクト名が異なる**
2. **URL構造が異なる**
3. **プロジェクトが別のアカウントまたはチームにある**

## 🔍 正しいURLを見つける方法

### 方法1: ダッシュボードから段階的にアクセス（推奨）

1. **Vercelダッシュボードにアクセス**
   ```
   https://vercel.com/dashboard
   ```

2. **プロジェクト一覧から「cmsbeppin」を探す**
   - プロジェクト一覧が表示される
   - 「cmsbeppin」または類似の名前を探す
   - 見つからない場合は、デプロイURLから推測

3. **プロジェクトをクリック**
   - プロジェクト名をクリックして、プロジェクトページに移動

4. **現在のURLを確認**
   - ブラウザのアドレスバーのURLを確認
   - 例えば: `https://vercel.com/[アカウント名]/cmsbeppin`
   - または: `https://vercel.com/cmsbeppin`

5. **Settingsを見つける**
   - プロジェクトページの上部にタブメニューがあるか確認
   - 「Settings」タブをクリック
   - または、URLの末尾に `/settings` を追加

6. **Environment Variablesを選択**
   - Settingsページで「Environment Variables」タブをクリック
   - または、URLの末尾に `/environment-variables` を追加

### 方法2: 検索機能を使用

1. **プロジェクトページで「F」キーを押す**
   - Quick Navigationが開く

2. **「Environment Variables」と検索**
   - 検索結果から選択

### 方法3: プロジェクト設定メニューから

1. **プロジェクトページで、以下の場所を確認:**
   - プロジェクト名の横の「...」アイコン
   - 右上の設定アイコン
   - 上部ナビゲーションバー

2. **「Settings」または「Project Settings」を選択**

## 📋 手順（ステップバイステップ）

### ステップ1: ダッシュボードにアクセス

1. **ブラウザで以下にアクセス:**
   ```
   https://vercel.com/dashboard
   ```

2. **ログイン（必要に応じて）**

### ステップ2: プロジェクトを見つける

1. **プロジェクト一覧を確認**
   - デプロイURLが `https://cmsbeppin.vercel.app` の場合
   - プロジェクト名は通常 `cmsbeppin`

2. **プロジェクトをクリック**
   - プロジェクト名またはカードをクリック

### ステップ3: プロジェクトページでSettingsを見つける

1. **プロジェクトページの上部を確認**
   - タブメニューがあるか確認
   - 「Overview」「Deployments」「Analytics」「Settings」など

2. **Settingsタブをクリック**
   - 見つからない場合は、以下の方法を試す

### ステップ4: Environment Variablesを見つける

1. **Settingsページに移動後**
   - 左サイドバーまたはメインエリアを確認
   - 「Environment Variables」または「環境変数」を探す

2. **クリックして開く**

## 🎯 最も確実な方法

**段階的にアクセスする:**

1. **ダッシュボード**: https://vercel.com/dashboard
2. **プロジェクトを選択**: プロジェクト一覧から「cmsbeppin」をクリック
3. **現在のURLを確認**: ブラウザのアドレスバーを見る
4. **URLの末尾に `/settings/environment-variables` を追加**
   - 例: `https://vercel.com/[現在のURL]/settings/environment-variables`

## ⚠️ プロジェクト名が異なる場合

プロジェクト名が「cmsbeppin」でない場合:

1. **デプロイURLから確認**
   - `https://cmsbeppin.vercel.app` の場合、プロジェクト名は `cmsbeppin`
   - 異なる場合は、そのプロジェクト名を使用

2. **プロジェクト一覧で確認**
   - ダッシュボードでプロジェクト名を確認

3. **現在のプロジェクト名を共有**
   - 環境変数を設定するために、正確なプロジェクト名が必要です

## 📝 現在の状況を確認

以下の情報を共有していただけると、より具体的な案内ができます：

1. **プロジェクト一覧で表示されるプロジェクト名**
2. **プロジェクトページのURL**
3. **プロジェクトページで表示されているタブメニュー**

これらの情報があれば、正確なURLを案内できます。

## 🚀 次のステップ

### 1. 環境変数の設定

`.env.local`ファイルを作成：

```env
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
JWT_SECRET=your_random_secret_at_least_32_characters
JWT_EXPIRES_IN=24h
```

### 2. データベースセットアップ

Supabaseダッシュボードで `supabase/point-card-schema.sql` を実行

### 3. 開発サーバーの起動

```bash
cd /Users/sawashimanoriyuki/point-card-pwa
npm run dev
```

### 4. ブラウザでアクセス

- `http://localhost:3000` - ホームページ
- `http://localhost:3000/register` - 顧客登録
- `http://localhost:3000/login` - 顧客ログイン
- `http://localhost:3000/admin/login` - 管理者ログイン

# 環境変数設定 - ステップバイステップガイド

## ✅ Environment Variablesの設定画面が見つかりました！

以下の手順で、3つの環境変数を設定してください。

## 📋 設定する環境変数

以下の3つの環境変数を追加します：

1. **VAPID_PUBLIC_KEY**（公開キー）
2. **VAPID_PRIVATE_KEY**（秘密キー）
3. **VAPID_MAILTO**（連絡先メールアドレス）

## 🚀 設定手順

### ステップ1: 最初の環境変数を追加

1. **「Create new」タブが選択されていることを確認**
   - 画面の上部に「Create new」と「Link Shared Environment Variables」のタブがある
   - 「Create new」タブが選択されている（下線が引かれている）

2. **「Key」欄に入力**
   - 以下のキー名を入力: `VAPID_PUBLIC_KEY`

3. **「Value」欄に入力**
   - 以下の値を入力:
   ```
   BBUKrYxJ-USIbUWVeH9bBW67b-a3o0_Sc0MOpVONQ9iy__l5py6HDORYHYlgjV5VxLHAIkbo8KwUHpVi4XmOEj0
   ```

4. **「Environments」ドロップダウンを確認**
   - 「All Environments」が選択されていることを確認
   - もし選択されていない場合は、「All Environments」を選択

5. **「Sensitive」トグルはオフのままでOK**
   - 公開キーなので、Sensitiveにする必要はありません

6. **「Add Another」ボタンをクリック**
   - 次の環境変数を追加するためにクリック

### ステップ2: 2つ目の環境変数を追加

1. **新しい「Key」欄が表示される**

2. **「Key」欄に入力**
   - 以下のキー名を入力: `VAPID_PRIVATE_KEY`

3. **「Value」欄に入力**
   - 以下の値を入力:
   ```
   uhvF8h2djTyKfCh0Hm8XrvGbhMCmimImg-paf44D_Ec
   ```

4. **「Environments」ドロップダウンを確認**
   - 「All Environments」が選択されていることを確認

5. **「Sensitive」トグルをONにする** ⚠️ 重要
   - 秘密キーなので、必ずSensitiveをONにしてください
   - これにより、値が隠され、後で読み取ることができなくなります

6. **「Add Another」ボタンをクリック**
   - 最後の環境変数を追加するためにクリック

### ステップ3: 3つ目の環境変数を追加

1. **新しい「Key」欄が表示される**

2. **「Key」欄に入力**
   - 以下のキー名を入力: `VAPID_MAILTO`

3. **「Value」欄に入力**
   - 以下の値を入力（適切なメールアドレスに変更してください）:
   ```
   admin@example.com
   ```
   ⚠️ 重要: `admin@example.com` を実際のメールアドレスに変更してください

4. **「Environments」ドロップダウンを確認**
   - 「All Environments」が選択されていることを確認

5. **「Sensitive」トグルはオフのままでOK**
   - メールアドレスは公開情報なので、Sensitiveにする必要はありません

### ステップ4: 保存

1. **3つの環境変数がすべて入力されていることを確認**
   - `VAPID_PUBLIC_KEY`
   - `VAPID_PRIVATE_KEY`
   - `VAPID_MAILTO`

2. **「Save」ボタンをクリック**
   - 画面の右下にある「Save」ボタンをクリック

3. **保存が完了するまで待つ**
   - 「Saved」または「Success」メッセージが表示されることを確認

4. **再デプロイが必要な場合**
   - 環境変数を追加した後、自動的に再デプロイがトリガーされる場合があります
   - または、「Deployments」タブから手動で再デプロイを実行する必要がある場合があります

## ✅ 設定完了の確認

1. **環境変数一覧を確認**
   - 設定した3つの環境変数が表示されていることを確認
   - 特に `VAPID_PRIVATE_KEY` は「●●●●●●●●」のように隠されていることを確認

2. **各環境変数の「Environments」を確認**
   - すべて「All Environments」が選択されていることを確認

## 📋 設定チェックリスト

- [ ] `VAPID_PUBLIC_KEY` が設定されている
- [ ] `VAPID_PRIVATE_KEY` が設定されている（Sensitive: ON）
- [ ] `VAPID_MAILTO` が設定されている（適切なメールアドレスに変更済み）
- [ ] すべての環境変数で「All Environments」が選択されている
- [ ] 「Save」ボタンをクリックして保存完了

## ⚠️ 重要な注意事項

1. **VAPID_PRIVATE_KEYは必ずSensitiveにする**
   - 秘密キーなので、必ずSensitiveトグルをONにしてください

2. **VAPID_MAILTOは適切なメールアドレスに変更**
   - `admin@example.com` を実際のメールアドレスに変更してください
   - 例: `your-email@example.com`

3. **環境変数の値に誤りがないか確認**
   - コピー&ペーストする際に、余分なスペースや改行が入らないように注意してください

## 🎉 設定完了後

環境変数の設定が完了したら、実装を続けることができます。

設定が完了したら、お知らせください。次のステップ（実装）に進みます！
